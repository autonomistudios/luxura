## Key Design Decisions (tell Gemini these explicitly)
1. Gold is NOT decorative — it signals active, selected, or data that matters
2. Cards use gradient (#111116→#0B0B0E), NOT flat color — creates subtle depth
3. Status badges are large and bold — DNA LOCKED is a statement, not a footnote
4. Hero numbers use Playfair italic at 56–72px — they should command the room
5. Agent pipeline strip makes the AI process VISIBLE — don't hide the pipeline
6. Atmospheric radial glows behind key elements (quota arc, forge grid, hero section)
7. Unselected cards dim to 45% brightness when one is selected (Visual DNA step)
8. Selected card description animates in — the copy is part of the interaction
