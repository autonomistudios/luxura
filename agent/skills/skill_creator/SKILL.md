# SKILL: Skill Creator

## Overview
This skill is responsible for generating new meta-skills and extending the capabilities of the agent ecosystem.

## Core Functions
1.  **Analyze Needs**: Identify missing functionalities in existing skills.
2.  **Generate SKILL.md**: Create the instruction and metadata template for a new skill.
3.  **Validate Structure**: Ensure new skills follow the defined organizational patterns.
4.  **Integration**: Plug new skills into the orchestration layer.

## Thresholds & Rules
- New skills must include a `SKILL.md` with clear directives.
- Follow the "DNA Fidelity" principle in all generated code.
