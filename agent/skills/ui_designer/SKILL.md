# SKILL: Mantis UI Designer

## Overview
Responsible for creating and maintaining the high-end, premium "Mantis" interface for the LuxAura Studio.

## Core Functions
1.  **Iterative UI Visualization**: Design the "Loop" state where users can see the Critic & Refinement progress in real-time.
2.  **Luxury Visual Language**: Use dark modes, glassmorphism, and gold/silver accents (Mantis brand colors).
3.  **Micro-animations**: Implement smooth transitions and "Agentic Pulse" animations to indicate AI thought processes.
4.  **Information Hierarchy**: Ensure that "Critic's Notes" and "Structural DNA" reports are front-and-center during the refinement phase.

## Thresholds & Rules
- **Aesthetics are Functional**: A premium look isn't optional; it's a core requirement for LUXury brand positioning.
- **Glassmorphism**: Use `backdrop-filter: blur()` extensively for panels.
- **Micro-interactions**: Every agent report update must have a subtle entry animation.
